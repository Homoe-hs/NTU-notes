# ARM vs. MIPS
## 比较
|                 | ARM                       | MIPS                       |
| --------------- | ------------------------- | -------------------------- |
| ISA             | RISC                      | dif RISC                   |
| num of register | 16                        | 32                         |
| bus             | support 64-bit data buses |                            |
| instructiong    | MOV instructions          | no eq inst | 

## 结论
- MIPS主要在嵌入式系统中使用
- 目前，ARM在业界比MIP更受欢迎
