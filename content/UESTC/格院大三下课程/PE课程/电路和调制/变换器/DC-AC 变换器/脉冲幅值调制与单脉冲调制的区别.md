| 区别               | 脉冲幅值调制            | 单脉冲调制         |
| ------------------ | ----------------------- | ------------------ |
| 直流侧电压（电流） | 需要改变 （晶闸管整流） | 不变（二极管整流） |
| 方波导通           | 时间固定                | 不固定             |
| 导通角             | 恒定                    | 调整               |
| 快速功率元件       | 不需要                  | 需要，例如IGBT                   |
