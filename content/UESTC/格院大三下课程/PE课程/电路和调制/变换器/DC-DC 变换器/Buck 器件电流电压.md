## 开关管 S

S 在导通时流过电流，关断时承受电压:

| S    | 导通        | 关断    |
| ---- | ----------- | ------- |
| 电压 | 0           | $V_{i}$ |
| 电流 | $i_{L_{1}}$ | 0        |

## 二极管 D

D 仅在 S 导通时承受反压，S关断时续流:

| D    | S 导通   | S 关断 |
| ---- | -------- | ------ |
| 电压 | $-V_{i}$ | 0      |
| 电流 | 0        | $i_{L_{1}}$       |
