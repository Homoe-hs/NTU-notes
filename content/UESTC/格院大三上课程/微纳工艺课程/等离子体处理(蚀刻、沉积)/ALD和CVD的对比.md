| 特性       | CVD                              | ALD                                    |
| ---------- | -------------------------------- | -------------------------------------- |
| 沉积温度   | 高                               | 中等                                   |
| 沉积机理   | 气相沉积，直接反应               | 气相沉积，表面化学反应                 |
| 膜厚控制   | 难以精确                         | 十分精确                               |
| 沉积均匀性 | 取决于反应室设计和反应气体流动性 | 沉积均匀                               |
| 沉积速率   | 高                               | 低                                     |
| 应用       | 大面积薄膜沉积                   | 器件制造中的原子级控制和纳米级薄膜沉积 |
|            |                                  |                                        |
