|            | SEM  | TEM                  | STM      | AFM              |
| ---------- | ---- | -------------------- | -------- | ---------------- |
| 横向分辨率 | 5 nm | 0.2 nm               | 0.1 nm   | 30 nm            |
| 纵向分辨率 | 无   | 无                   | 0.01 nm  | 0.1 nm           |
| 放大类型   | 2D   | 2D                   | 3D       | 3D               |
| 样品准备   | 无   | 困难（FIB，MIlling） | 超净表面 | 干净表面         |
| 环境要求   | 真空 | 真空                 | 真空     | 真空、空气、液体 |
| 横截面图像 | 有   | 有                   | 无       | 无                 |
